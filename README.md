# Hexagon

Monorepo: strona freelancera IT + backend chatbota (w przygotowaniu).

## Struktura

```
Hexagon/
├── client/   # Next.js (frontend)
└── server/   # FastAPI + LangGraph (backend, wkrótce)
```

## Client (Next.js)

```bash
# z katalogu głównego
npm run dev

# lub bezpośrednio
cd client && npm run dev
```

Strona: [http://localhost:3000](http://localhost:3000)

## Server (FastAPI)

Backend chatbota — folder przygotowany, implementacja w osobnym kroku.
Zobacz [server/README.md](./server/README.md).
